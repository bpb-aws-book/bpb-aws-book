# Python Coding Standards

## File Header

Every `.py` file must include the following comment header at the top of the file:

```python
# File: <filename>.py
# Description: <brief description of the file's purpose>
# Author: <author name>
# Created: <YYYY-MM-DD>
```

- `File` — the file name including extension
- `Description` — a concise one-line summary of what the file does
- `Author` — the name of the person or agent who created the file
- `Created` — the date the file was first created in `YYYY-MM-DD` format
